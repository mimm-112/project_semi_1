# 발표 PPT용 시각화 안내

두 개의 시각화 코드로 총 9개 그래프를 생성합니다.
`그래프이미지/` 폴더에 미리 생성된 PNG가 있고, 코드를 실행하면 재생성됩니다.

---

## 실행 방법

```bash
pip install pandas numpy matplotlib scipy
# 같은 폴더에 merged_full.csv, drozy_kss_pvt.csv, state_2x2.csv 필요
python eda_visualization.py      # fig1~6 생성
python ml_comparison_viz.py      # ml1~3 생성
```

한글 폰트는 Noto Sans CJK를 자동 사용합니다.
다른 환경에서 한글이 깨지면 코드 상단 폰트 경로를 수정하세요.

## 수치 출처

- EDA 그래프: merged_full.csv, drozy_kss_pvt.csv, state_2x2.csv에서 직접 계산
- ML 그래프: 03b_model_comparison.py의 5-fold CV 결과 (AUC)
  - 수치를 갱신하려면 03b를 재실행 후 ml_comparison_viz.py의 값 수정
- 알고리즘 비교 성능:
  - Logistic 0.746 / AdaBoost 0.777 / LightGBM 0.772 / XGBoost 0.770
  - RandomForest 0.792 / GradientBoosting 0.806 (최종)

## PPT 제작 팁

- fig4(2×2)와 fig5(숨은위험 검증)는 발표의 핵심 → 크게 배치
- ml1(알고리즘 비교)은 "여러 모델을 비교했다"는 근거로 모델설계 페이지에 필수
- 색상 일관성: 객관=파랑, 주관=보라, 위험=빨강, 정상=초록 계열로 통일됨
- 모든 그래프는 dpi 120으로 저장되어 슬라이드 확대 시에도 선명
