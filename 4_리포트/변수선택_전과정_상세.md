우리 데이터의 변수는 성격이 완전히 다른 두 그룹입니다. 선택 방법도 다릅니다.

- **(A) PVT 파생변수** — 반응속도 검사 원시데이터에서 우리가 직접 계산한 지표

(A)는 "서로 겹치는 문제(공선성)"를 풀어야 하고,

# 파트 A. PVT 파생변수 선택 (13개 → 5개)

## A-1. 먼저, 파생변수 13개는 어떻게 나왔나

PVT 원시데이터는 "자극이 언제 떴고(onset), 얼마 만에 반응했는지(response_time)"
두 정보뿐입니다. 여기서 다음 13개 지표를 계산했습니다.

| # | 컬럼명 | 의미 |
|---|---|---|
| 1 | `n_trial` | 5분간 제시된 총 자극 수 |
| 2 | `n_valid` | 유효 반응 수 (100~500ms) |
| 3 | `false_starts` | 성급한 반응 수 (100ms 미만, 자극 전에 누름) |
| 4 | `lapses` | 주의 실패 수 (500ms 이상) |
| 5 | `lapse_rate` | 주의 실패 비율 (lapses ÷ n_trial) |
| 6 | `mean_rt` | 평균 반응시간 (ms) |
| 7 | `median_rt` | 중앙 반응시간 (ms) |
| 8 | `sd_rt` | 반응시간 표준편차 (변동성) |
| 9 | `min_rt` | 최소 반응시간 (가장 빠른 반응) |
| 10 | `max_rt` | 최대 반응시간 (가장 느린 반응) |
| 11 | `mean_speed` | 평균 반응속도 (1/RT의 평균) |
| 12 | `slowest10_rt` | 가장 느린 10% 반응의 평균 |
| 13 | `cv_rt` | 변동계수 (sd_rt ÷ mean_rt) |

## A-2. 왜 13개를 다 못 쓰나 — 공선성(당연한 현상)

이 13개는 **같은 반응시간 데이터를 여러 방식으로 요약한 것**입니다.
그래서 서로 겹치는 게 당연합니다. 예를 들어:

- `mean_speed`는 `mean_rt`의 역수(1/RT) → 사실상 같은 정보
- `lapse_rate`는 `lapses`를 `n_trial`로 나눈 것 → 같은 정보
- `n_trial` = `n_valid` + `false_starts` + `lapses` → 정의상 완전 종속

이렇게 겹치는 변수를 모델에 다 넣으면 "다중공선성" 문제가 생깁니다.
→ 모델이 어느 변수가 진짜 중요한지 구분 못 하고, 계수가 불안정해집니다.

## A-3. 실제로 얼마나 겹치는가 — 상관계수 (|r|≥0.8)

13개 변수끼리 상관을 본 결과, 강하게 겹치는 쌍들:

| 변수 쌍 | 상관계수 r |
|---|---|
| `lapses` ↔ `lapse_rate` | +1.00 (완전 동일) |
| `mean_rt` ↔ `median_rt` | +0.98 |
| `mean_rt` ↔ `mean_speed` | −0.98 (역수 관계) |
| `median_rt` ↔ `mean_speed` | −0.98 |
| `mean_rt` ↔ `slowest10_rt` | +0.90 |
| `sd_rt` ↔ `cv_rt` | +0.88 |
| `mean_speed` ↔ `slowest10_rt` | −0.86 |
| `median_rt` ↔ `slowest10_rt` | +0.84 |
| `n_trial` ↔ `n_valid` | +0.81 |
| `sd_rt` ↔ `max_rt` | +0.81 |

## A-4. VIF로 공선성 정량 확인 (선정 전)

VIF(분산팽창인자)는 "이 변수가 다른 변수들로 얼마나 설명되는가"를 나타냅니다.
**VIF > 10이면 심각한 공선성**입니다.

| 컬럼명 | VIF | 판정 |
|---|---|---|
| `n_trial`, `n_valid`, `false_starts`, `lapses` | ∞ (무한대) | 정의상 완전 종속 |
| `lapse_rate` | 356.6 | 심각 |
| `mean_rt` | 179.9 | 심각 |
| `sd_rt` | 170.8 | 심각 |
| `cv_rt` | 119.7 | 심각 |
| `median_rt` | 71.6 | 심각 |
| `mean_speed` | 46.0 | 심각 |
| `slowest10_rt` | 20.9 | 심각 |
| `max_rt` | 6.8 | 양호 |
| `min_rt` | 5.3 | 양호 |

→ 13개 중 대부분이 VIF 10을 넘음. 정리 필요.

## A-5. 겹치는 변수를 '축(그룹)'으로 묶기

상관 결과를 보면, 13개는 사실 **5개의 독립적인 정보 축**으로 압축됩니다.

| 축 | 포함 변수 | 이 축이 재는 것 |
|---|---|---|
| ① 반응속도 | `mean_rt`, `median_rt`, `mean_speed`, `slowest10_rt`, `min_rt`, `max_rt` | 얼마나 빠른가 |
| ② 변동성 | `sd_rt`, `cv_rt` | 얼마나 들쭉날쭉한가 |
| ③ 주의실패 | `lapses`, `lapse_rate` | 얼마나 놓치는가 |
| ④ 반응량 | `n_valid`, `n_trial` | 얼마나 반응했나 |
| ⑤ 성급함 | `false_starts` | 성급하게 누르나 |

## A-6. 각 축에서 대표 1개 선정 → 최종 5개

| 축 | 선정 변수 | 선정 이유 |
|---|---|---|
| ① 반응속도 | `mean_rt` | PVT 연구의 가장 표준적 지표 |
| ② 변동성 | `sd_rt` | 해석이 직관적(표준편차) |
| ③ 주의실패 | `lapses` | 피로 연구의 대표 지표(문헌 표준) |
| ④ 반응량 | `n_valid` | 유효 반응 수 |
| ⑤ 성급함 | `false_starts` | 경계 이완 지표 |

## A-7. 선정 후 VIF 재확인 (문제 해결 입증)

| 컬럼명 | VIF |
|---|---|
| `mean_rt` | 1.4 |
| `sd_rt` | 1.3 |
| `lapses` | 1.3 |
| `n_valid` | 1.2 |
| `false_starts` | 1.1 |

→ **전부 2 미만.** 공선성 완전히 해결. 이 5개를 PVT 최종 변수로 확정.

# 파트 C. PVT를 주관피로 예측에서 뺀 이유 (중요)

PVT 5개 변수를 타깃(SicknessQ)과 상관 검정하면:

| 컬럼명 | r | p | 통계기준 |
|---|---|---|---|
| `mean_rt` | −0.151 | 0.050 | 미달 |
| `sd_rt` | +0.023 | 0.771 | 미달 |
| `lapses` | −0.084 | 0.280 | 미달 |
| `n_valid` | +0.141 | 0.067 | 미달 |
| `false_starts` | −0.014 | 0.861 | 미달 |

→ **PVT는 주관피로(SicknessQ)와 통계적으로 무관.**
→ 따라서 "주관피로 예측"의 입력에서는 PVT 제외.
→ 그러나 PVT는 **선행연구가 검증한 객관적 각성 지표**이므로,
   통계 상관이 아닌 **도메인 근거**로 별도의 객관 축을 구성.
→ 이 "PVT와 주관피로의 독립성"이 바로 2×2 구조(숨은 위험 탐지)의 근거.
