# 수면·피로 데이터 분석 리포트 (DA 파트)

> 이 리포트는 통계를 잘 모르는 분도 이해할 수 있도록 용어를 풀어서 작성했습니다.

---

## 0. 이 실험이 어떤 것인지 (매우 중요 — 먼저 읽어주세요)

**데이터 출처:** OpenNeuro ds000201, "스톡홀름 Sleepy Brain 연구" (스웨덴 카롤린스카 연구소)

**실험이 실제로 한 일:**
- 참가자에게 **하룻밤에 3시간만 자게 함** (밤 끝 무렵 3시간 수면 기회를 줌)
  - ※ "평소보다 3시간 덜 잔 것"이 아니라 **"총 3시간만 잔 것"**입니다.
  - ※ 밤새 안 재운 "완전 수면박탈"이 아니라 하룻밤 **"부분 수면박탈"**입니다.
- 같은 사람이 **정상수면(충분히 잔 밤)과 부분수면박탈(3시간만 잔 밤)을 둘 다** 경험
  (이런 방식을 "교차설계"라고 함 — 같은 사람을 두 조건에서 비교하므로 개인차 영향이 적음)

**참가자:**
- 총 86명. 두 연령대: 젊은층(20–30세), 노인층(65–75세)
- 모두 건강한 사람 (정신·신경 질환, 고혈압·당뇨 없음, 향정신성 약물 미복용)

**주의할 점:**
- 이 연구의 원래 목적은 "수면박탈이 **감정 처리**에 미치는 영향"이었습니다.
- PVT(반응속도 검사)는 부수적으로만 측정되었습니다.
- → 그래서 PVT에는 수면박탈 신호가 약하게 나타납니다 (뒤에서 설명).

## 0-1. 측정 시점 (매우 중요 — 각 변수가 언제 측정됐는지)

각 변수가 수면박탈의 "언제"를 반영하는지 알아야 결과 해석이 가능합니다.

**전체 타임라인:**

```
[집에서] 대조 밤: 정상 수면          [집에서] 실험 밤: 3시간만 수면
   PSG(수면다원검사)로 수면 기록          PSG로 수면 기록
        │                                     │
        │   (각 밤의 "다음 날")                │
        ▼                                     ▼
   ───────── 저녁 검사 세션 (오후 5시~8시 시작) ─────────
        ├─ 검사 직전: PVT (5분 반응속도 검사)
        ├─ 스캔 중: SicknessQ, PANAS 등 주관 설문 (byScanner)
        └─ MRI 스캔 (감정 얼굴 과제)

   ※ 두 세션(정상/박탈)은 약 한 달 간격, 순서는 무작위
```

**핵심 — 모든 검사는 "수면박탈 다음 날 저녁"에 측정됨:**
- 참가자는 집에서 3시간만 자고 → 하루를 보낸 뒤 → **저녁(오후 5~8시)에 검사**
- "밤새우고 새벽에 바로 검사"가 **아님**. 3시간 자고 + 하루 지난 후 + 저녁 측정
- PVT, SicknessQ 등 우리가 쓰는 모든 변수 = 이 저녁 세션에서 측정됨

**이것이 분석 결과에 주는 의미:**
- 저녁 시간대(오후 5~8시)는 일주기 리듬상 **각성도가 회복되는 시간**
- 밤샘 직후 새벽이었다면 PVT가 크게 나빠졌겠지만,
  3시간 수면 + 하루 경과 + 각성 높은 저녁 측정 → **PVT에 흔적이 적게 남음**
- 반면 주관적 피로감(SicknessQ)에는 흔적이 남음
- → 이 측정 시점이 "PVT는 약한데 주관지표는 반응하는" 현상을 설명함

**변수별 측정 시점 요약:**

| 변수 | 측정 시점 | 성격 |
|---|---|---|
| PVT 파생변수 | 저녁 세션 중, 스캔 직전 (5분) | 그 세션의 상태 |
| SicknessQ (byScanner) | 저녁 세션 중, 스캔 시점 | 그 세션의 상태 |
| PANAS (byScanner) | 저녁 세션 중, 스캔 시점 | 그 세션의 상태 |
| 수면시간(PSG) | 검사 전날 밤, 집에서 | 그 세션의 수면량 |
| ESS·ISI·성격 등 | 사전 스크리닝 (1회) | 개인 고정 특성 |

## 1. 우리가 풀려는 문제

**목표:** 여러 측정값으로 그 사람의 "피로 상태"를 맞히는 모델 만들기

- **종속변수 y (맞히려는 것):** SicknessQ 점수
  - 각 검사 직후 참가자가 스스로 매긴 "몸이 아프고 무거운 느낌" 설문 점수
  - 이 점수를 중앙값(가운데 값) 기준으로 잘라 **"피로 높음 / 피로 낮음" 두 그룹**으로 나눔
- **독립변수 X (맞히는 데 쓰는 재료) — 총 11개:**
  - PVT(반응속도 검사)에서 뽑은 5개:
    - 반응시간(평균), 반응 변동성, 주의 놓침 횟수, 유효 반응 수, 성급한 반응 수
  - 개인 특성 6개:
    - 나이, 성별, 주간졸림(ESS), 불면(ISI), 신경증 성향, 특성불안(STAI)

**왜 SicknessQ를 타깃으로 골랐나:**
- PVT 위험도를 타깃으로 쓰면 "PVT로 만든 답을 PVT로 맞히는" 순환이 생김 → 제외
- SicknessQ는 PVT와 무관한 별도 설문이고, 수면박탈에 확실히 반응함 → 적합

## 2. 데이터 준비 과정 (DA 작업 순서)

1. **PVT 원시 데이터 정제**
   - 참가자마다 검사 길이가 달라(41~98회) → **모두 처음 5분(300초)으로 통일**
   - 파일 오류(줄바꿈 깨짐) 수정, 비정상 반응 제거
2. **PVT 파생변수 생성** — 반응속도 데이터에서 12개 지표 계산
3. **다중공선성(VIF) 점검**
   - "다중공선성"이란: 변수들끼리 너무 비슷해서 중복되는 현상
   - PVT 지표 12개가 서로 거의 같은 정보를 담고 있었음 (예: 평균반응시간과 반응속도는 사실상 같은 것의 앞뒤 관계)
   - → 겹치지 않는 **대표 5개만 선별**
4. **설문 데이터 결합** — 위 데이터에 참가자 설문 전체를 붙임

---

## 3. 핵심 발견

### 발견 1: 수면박탈은 반응속도(PVT)를 거의 못 바꾼다
- 정상수면 vs 부분수면박탈에서 PVT 평균반응시간 차이: 거의 없음 (통계적으로 무의미)
- 이유: 3시간이라도 잤기 때문에(부분 박탈), 저녁 시간대에 측정했기 때문에

### 발견 2: 그러나 "주관적 느낌"은 확실히 나빠진다
- 못 잤을 때 "아픈 느낌(SicknessQ)"과 "부정 기분"이 뚜렷하게 증가
- → **부분 수면박탈은 '수행 능력'보다 '주관적 컨디션'을 먼저 무너뜨린다**

### 발견 3: 개인차가 매우 크다
- 평균으로는 효과가 없어 보이지만, 사람마다 반응이 크게 다름
- 어떤 사람은 못 자면 크게 느려지고(취약), 어떤 사람은 오히려 멀쩡함(저항)
- 단, 이 취약성이 "누구인지"는 우리가 가진 정보로 예측 불가

---

## 4. 예측 모델 결과

### 4-1. 여러 알고리즘 비교 (판별 성능 = AUC)
> **AUC란:** 0.5면 찍기(동전던지기), 1.0이면 완벽. 0.7 이상이면 쓸만함, 0.8 이상이면 좋음.

| 모델 | AUC | 설명 |
|---|---|---|
| 로지스틱 회귀 | 0.746 | 가장 단순한 기본 모델 |
| 랜덤포레스트 (기본) | 0.775 | 여러 결정트리를 합친 모델 |
| **GradientBoosting (튜닝)** | **0.806** | **최종 선정 — 가장 우수** |

- 하이퍼파라미터 튜닝(모델 설정 최적화)으로 0.746 → 0.806까지 향상

### 4-2. PVT만 쓰면? vs 개인특성까지 쓰면?

| 사용 재료 | AUC |
|---|---|
| PVT만 | 0.56 (거의 찍기 수준) |
| PVT + 개인특성 | 0.75~0.81 |

→ **피로 판별력의 대부분은 PVT가 아니라 개인특성(특히 불안)에서 나온다**

### 4-3. 무엇이 피로 판별에 중요한가 (최종 모델 기준)

| 순위 | 변수 | 중요도 |
|---|---|---|
| 1 | 특성불안(STAI) | 0.30 |
| 2 | PVT 반응시간 | 0.23 |
| 3 | PVT 변동성 | 0.14 |
| 4 | PVT 반응수 | 0.08 |

- 불안 성향이 높은 사람이 피로를 더 많이 호소
- 트리 모델에서는 PVT도 의미 있게 기여 (불안과의 조합으로 작동)

---

## 5. 결론과 한계

**결론:**
- PVT + 개인특성으로 피로 상태를 AUC 0.81 수준으로 판별 가능
- 최종 모델: GradientBoosting (튜닝 완료)

**정직한 한계:**
- 이 데이터는 원래 감정 연구용이라 PVT 신호가 약함
- 표본이 크지 않음(169세션)
- 주관적 피로는 본질적으로 노이즈가 많음
- → 이 한계들은 데이터 특성이며, 발표 시 솔직하게 고찰로 다루는 것이 바람직

**서비스 함의:**
- "PVT만으로 피로 판별"은 어려움
- 개인 특성(불안·불면 등)을 함께 입력받아야 판별력이 확보됨
